# 领土战争
查看全部战争相关指令：`/wars help`。
详细内容请看https://wiki.incredibleplugins.com/lands/players/general/wars 如果本文与wiki有冲突则以wiki为准。

本章基于上一章中领土部分的进阶内容。
## 开战条件
1. 成员要求：
    - 攻击方领土：必须拥有至少 3​ 位信任成员。
    - 防御方领土：必须拥有至少 5​ 位信任成员。
2. 存款要求：
    - 攻击方领土：存款余额必须大于 50,000。
    - 防御方领土：存款余额必须大于 10,000。
3. 存续时间要求：
    - 攻击方领土：必须成立超过 3​ 天。
    - 防御方领土：必须成立超过 5​ 天。

## 宣战流程
攻击方需在 **每日早上9点至晚上11点​** 之间，使用指令`/wars declare <land>`向目标领土发出战争宣言。</br>
宣言发出时，防御方领土必须有成员在线。</br>
防御方需在 **5天内​** 回应并接受宣战。若逾期未接受，则视为拒绝，防御方不会受到任何惩罚，此举是为了防止单方面宣战。

## 战争中
持续时间：**战争将持续 36小时（1.5天）。**</br>
系统提示：战争期间，系统会每隔 15分钟​ 向所有领土成员发送通知。</br>

规则变更：战争期间，在敌对领土内，以下限制将被解除：</br>
可以放置和破坏藤蔓、梯子、脚手架。</br>
无视对方的物品拾取限制、玩家攻击限制以及领土进入限制。</br>

## 胜负与战果
- 攻击方投降：扣除其领土存款的 50%​ 给予防御方。
- 防御方投降：扣除其领土存款的 30%​ 给予攻击方。
- 获胜方将获得对方领土存款的 75%。

## 战后
- 投降方：获得持续4天​的“战争护盾”（在此期间无法被宣战）。
- 交战双方：无论胜负，战后均获得持续 7天​ 的“战争护盾”。

## 占领机制
战争采用积分制。使用`/war menu`可以合成“占领旗帜”，用于占领敌方区块以获得分数。</br>
占领流程：在目标区块放置旗帜后，需有己方成员在该区块停留 10分钟​ 才算占领成功。占领期间，系统会每隔 30秒​ 用烟花特效提示周围玩家。</br>
旗帜归属：战争结束后，占领旗帜默认会归还给原领土。

# 区块漏斗
**Upgradeable Hoppers** 是一个 Minecraft 服务器插件（Spigot/Bukkit），其核心功能是**用单个高性能漏斗替代复杂的漏斗链**，从而大幅减少服务器因实体漏斗产生的 Tick 延迟（TPS Lag）。

**区块级吸取（Chunk Suction）**：漏斗可以设置为“区块吸取”模式，自动拾取整个区块（或指定半径）内掉落在地面上的物品，无需使用水流或漏斗矿车收集。

---

由于是测试,价格会进行调整,不是最终结果。</br>
只能在自己的领土或被信任的领土中使用。领土中被信任的其他人或领土拥有者可以编辑区块漏斗的设置</br>

主要命令:`/uhoppers help`</br>
查看管理已放置的升级漏斗命令:`/uhoppers list`</br>
购买漏斗命令:`/uhoppers get` 默认normal或chunk</br>
左键升级漏斗打开交互菜单,右键打开漏斗,可潜行时用工具破坏回收已放置的升级漏斗.

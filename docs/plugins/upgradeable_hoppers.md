# 区块漏斗
::: tip 提示
使用`/uhoppers help`获得这个插件的命令的用法。
:::

## 区块漏斗是什么？
**Upgradeable Hoppers** 是一款为 Minecraft 服务器（Spigot/Bukkit）设计的优化插件。其核心功能是**用单个高性能漏斗替代复杂的漏斗链**，从而显著降低服务器因实体漏斗产生的 Tick 延迟（TPS Lag），提升服务器性能。

<mark>这个插件目前并没有官方中文名，我们暂称为区块漏斗，但“升级漏斗”这样的名字也是这个插件。</mark>

## 核心特性
- 自动拾取整个区块内掉落在地面上的物品
- 无需依赖水流或漏斗矿车进行物品收集
- 支持在自己的领土或被信任的领土中使用
- 领土中被信任的其他成员或领土拥有者可以编辑区块漏斗的设置

## 操作
`/uhoppers list` - 查看并管理已放置的升级漏斗

`/uhoppers get` - 购买漏斗（默认类型：normal 或 chunk）

**左键点击区块漏斗** - 打开交互菜单

**右键点击升级漏斗** - 打开漏斗界面

**潜行 + 使用工具破坏** - 回收已放置的升级漏斗

## 注意事项
- 当前版本为测试阶段，漏斗价格可能会调整，并非最终定价
- 漏斗只能在**自己的领土**或**被信任的领土**中使用
- 漏斗功能在权限范围内自动运行，无需玩家手动干预收集流程

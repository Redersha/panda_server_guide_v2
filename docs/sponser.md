# 赞助
**赞助是获得不影响游戏平衡的游戏权限的重要途径**，主要包括三种方式：直接购买点券、购买月卡（减负包）以及参与众筹。

* 点券是用于购买权限的主要代币，**可兑换游戏币、装饰性或功能性的权限（如彩色聊天字体、更多附魔查询、称号装饰、增加上架物品数量等）**，有效期通常为一个月。  

* **月卡（减负包）为按月订阅服务**，购买后可一次性获得200点券，并每日领取800游戏币和一个不死图腾；在有效期内，还能享受圈地价格降低0.1、领地上限+1，以及mcmmo经验获取增加10%的福利。  

* 众筹则在服务器需要添加新插件时开放，**由多名玩家共同赞助以筹集所需金额**。参与众筹的玩家将加入众筹组，获得额外的领地及设家上限，并解锁部分娱乐和功能性插件权限（例如CMI的装饰性功能，如`/ride`、`/hat`、盔甲架编辑等），**这些权限均以娱乐和便利为主，不会影响玩家间的游戏平衡。**

前两种可以在游戏中输入`/菜单`后，于界面右下角点击赞助菜单进入相关界面。

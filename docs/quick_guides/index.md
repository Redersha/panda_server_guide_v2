# 快速指南

**请先阅读["快速指南 - 0. 安装和开始"](./0_install_and_start.md)。**

本章以尽量简短的语言、分游戏阶段（从下载、游戏初期、中后期发展等方面）指引玩家。

本章是一条线的，不会涉及全部玩法和教程。

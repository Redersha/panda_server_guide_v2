# 安装和开始
::: tip 提示
**加入qq群 276051457**
:::
## 1. 下载和安装启动器
**推荐使用[PCL启动器](https://ifdian.net/p/0164034c016c11ebafcb52540025c377)**

* 如果不能打开，请下载并安装[运行环境](https://ltcat.lanzouv.com/i1mhD2drlvpa)

更多有关PCL的错误，可以在群里[问群友](/docs/other/asking_requirements.md)或者查看[PCL常见问题](https://shimo.im/docs/qKPttVvXKqPD8YDC)
## 2. 下载整合包/原版游戏
**本服是插件服，不需要玩家安装任何模组，但是可以安装一些能改良原版游戏体验的模组。**

* 推荐使用[已经配置好的整合包（开箱即用）](https://github.com/Redersha/Minecraft-JE-Panda-Server-Cilent-Modpack)。
* 或者去PCL启动器的下载页面下载一个性能优化整合包

## 3. 将本服添加到服务器列表并开始游戏
1. 启动游戏
2. 点击‘多人游戏’
3. 点击‘添加服务器’
4. 填入服务器ip（需要你先加入qq群以获得ip）
## 4. 为账号设置登录密码
1. 进入服务器
2. 输入`/reg <你的密码> <你的密码>`并回车

**服务器用玩家ID来存储玩家信息**
## 5. 开始自由探索！
